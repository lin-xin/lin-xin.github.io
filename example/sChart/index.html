# 指南

# 介绍

sChart.js：一个小型简单的图表库。基于 HTML5 canvas 实现，无任何依赖，并且兼容移动端。

当前版本：V3.0.0

# 快速上手

可以使用 npm 安装

npm install schart.js

# 支持 es6 引入

import Schart from 'schart.js';

new Schart(canvasId, options);

# 支持script标签引入

<script src="schart.js"></script>
<script>
    new Schart(canvasId, options);
</script>
