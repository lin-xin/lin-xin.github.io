# Guide

# introduce

sChart.js: Small & simple HTML5 charts. Based on the implementation of HTML5 canvas, it has no dependency and is compatible with the mobile terminal.

Version: V3.0.0

# quick start

Install with NPM

npm install schart.js

# Support ES6

import Schart from 'schart.js';

new Schart(canvasId, options);

# Support for script tags

<script src="schart.js"></script>
<script>
    new Schart(canvasId, options);
</script>
