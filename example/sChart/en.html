Getting started

Using sChart.js

First, we need to introduce the Chart.js file into the page.

<script src="sChart.js"></script>
Create canvas

To create a chart, you first generate a canvas and instantiate an icon object through the canvas's ID.

Create chart

Generates diagrams by instantiating sChart objects.

new sChart(canvasId, type, data, options)

After you have completed the instantiation, you can see the generated chart on the canvas.

Bar chart

Use case
Data structure
var data = [
    {name:'2012', value:1141},
    {name:'2013', value:1499},
    {name:'2014', value:2260},
    {name:'2015', value:1170},
    {name:'2016', value:970},
    {name:'2017', value:1450}
]
Create chart
new sChart('canvas', 'bar', data, {
    title: 'Total sales of stores in recent years',
    bgColor: '#829dda',
    titleColor: '#ffffff',
    fillColor: '#72f6ff',
    axisColor: '#eeeeee',
    contentColor: '#bbbbbb'
});
Chart parameter
options = {
    padding: 50,                    // Inside margin of chart
    yEqual: 5,                      // The number of points on the Y axis
    bgColor: '#ffffff',             // The background color of chart
    fillColor: '#1E9FFF',           // The color of bar chart
    axisColor: '#666666',           // The color of the coordinate axis
    contentColor: '#eeeeee',        // The color of the horizontal lines of the content
    title: '',                      // The title of chart
    titleColor: '#000000',          // Title Color
    titlePosition: 'top'            // Title position: top / bottom
}

Pie chart

Use case
Data structure
var data = [
    {name:'T-shirt', value:1200},
    {name:'trousers', value:1222},
    {name:'dress', value:1283},
    {name:'coat', value:1314},
    {name:'jackets', value:2314}
]
Create chart
new sChart('canvas', 'pie', data, {
    title: 'Annual sales of goods in stores',
    bgColor: '#829dca',
    titleColor: '#ffffff',
    legendColor: '#ffffff'
});
Chart parameter
options = {
    bgColor: '#ffffff',             // The background color of chart
    title: '',                      // The title of chart
    titleColor: '#000000',          // Title Color
    titlePosition: 'top'            // Title position: top / bottom
    legendColor: '#000000',         // The color of the legend text
    legendTop: 40,                  // Legend position of top
    colorList: ['#1E9FFF', '#13CE66', '#F7BA2A', '#FF4949', '#72f6ff'],    // The color list of a pie chart
    radius: 100,                    // The radius of a pie chart
}
